# RAG/ML

Эта папка предназначена для RAG/ML команды.

## Структура
- `kb/` — локальные docpacks (документация и best practices) для RAG.

MVP-идея: документация хранится локально в виде `.md/.txt` файлов, разбитых на главы. Индексатор/ретривер читает все языки через единый интерфейс (см. `kb/README.md`).
